#  Dashboard Financeiro - Análise de Receita, Despesas e Margem

##  Objetivo
Desenvolver um dashboard para análise financeira com foco em geração de insights estratégicos sobre receita, custos e margem de lucro.


##  Principais Perguntas Respondidas
- Quais são os principais componentes da receita?
- Onde estão concentrados os maiores custos?
- Como a margem de lucro se comporta ao longo do tempo?
- Existem segmentos mais rentáveis?

##  Principais Insights
- Vendas representam a maior parcela da receita total
- Alguns custos têm impacto desproporcional na margem
- Segmentação revelou grupos com maior potencial de valor

##  Ferramentas
- Power BI
- DAX
- Excel / CSV
